# Changelog

## 0.1.0

- Initial release.
- Baseline: WCAG 2.2 AA for supported user journeys.
- Violations table: 6 ERROR covering non-semantic clickables, missing accessible names, dialogs without focus traps, hidden focus indicators, unchecked reduced-motion on auto-advancing carousels, and unannounced required status updates.
- Sections: Semantics, Keyboard, Focus Management, Forms and Errors, Dynamic Content, Motion/Contrast/Targets.
